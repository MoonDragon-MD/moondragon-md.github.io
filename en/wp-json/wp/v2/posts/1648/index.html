I happened to want to save an image from a website with avif format (android 12 technology), but then not be able to view it with default image reader. So I created a Nautilus script to convert it to png<\/p>\r\n\r\n

 <\/p>\r\n\r\n

 <\/p>\r\n\r\n

https:\/\/github.com\/link-u\/davif\/releases\/download\/v0.1.0-20200331\/davif_0.1.0-20200331_amd64.deb<\/a><\/p>\r\n\r\n

 <\/p>\r\n\r\n

Now download my script here:<\/p>\r\n\r\n

 <\/p>\r\n\r\n

https:\/\/www.dropbox.com\/s\/bu77yrq0h2kv8ik\/Avif-to-png.sh?dl=0<\/a><\/p>\r\n\r\n

 <\/p>\r\n\r\n

Now just move it to your home, full path:<\/p>\r\n\r\n

 <\/p>\r\n\r\n

\/home\/USER\/.local\/share\/nautilus\/scripts<\/p>\r\n\r\n

 <\/p>\r\n\r\n

Where instead of USER you will put the login name<\/p>\r\n\r\n

 <\/p>\r\n\r\n

Then give it execution permissions (right click > properties > permissions > allow execution)<\/p>\r\n\r\n
