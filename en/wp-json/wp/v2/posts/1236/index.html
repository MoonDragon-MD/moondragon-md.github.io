\nHello, everyone,
\nI was able to hear and make calls from the mobile phone connected in BT (bluetooth) through the computer with Ubuntu and with headphones with microphone connected to the PC.
\nLet’s say how modern cars that have the Bluetooth handsfree, in fact you can make calls without headphones inserted.<\/p>\n

0)<\/strong> Connect and pair the android smartphone (I used android 7.1) with BT to Ubuntu 18.04
\n1)<\/strong> Install ofono from package manager
\n2)<\/strong>Edit the file with<\/p>\n

sudo gedit \/etc\/dbus-1\/system.d\/ofono.conf<\/p>\n

putting in the final part<\/p>\n

\u00a0 <policy user=\"pulse\">
\n<allow send_destination=\"org.ofono\"\/>
\n<\/policy><\/code><\/p>\n

<\/busconfig><\/p>\n

3)<\/strong> edit the file with<\/p>\n

sudo gedit \/etc\/pulse\/default.pa<\/p>\n

Replace<\/p>\n

load-module module-bluetooth-discover<\/p>\n

with<\/p>\n

load-module module-bluetooth-discover headset=ofono<\/p>\n

4)<\/strong> Restart the PC
\n5)<\/strong> From smartphone go to BT and connect with a simple click to the already associated pc
\n6)<\/strong> In audio settings (system PC), in the input sub-menu, do not select the BT of the cell but the input of the microphone or of the PC or of the headphones connected to the PC (if it does not appear, change this setting during the call).
\n7)<\/strong> make a call and see if everything works! (remember when changing the microphone input as step 6)<\/p>\n

Since searching on the internet I have not found any solution I have worked hard to write this guide
\nI hope you can use it
