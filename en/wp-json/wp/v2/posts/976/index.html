\nThe problem jumps out as an unsuccessful update called ttf-mscorefonts-installer ie fonts (fonts) used by Microsoft, it is better to install them to have compatibility on World documents in Libre Office.
\nReturning to the problem here is how to solve the problem on the fly without long tricks:
\nLet’s open a terminal with ctrl + alt + t or by looking it up in the main menu and make the copy paste of<\/p>\n

sudo sed -i ‘s#http:\/\/downloads.sourceforge.net\/corefonts#https:\/\/github.com\/pushcx\/corefonts\/raw\/master#g’ \/usr\/share\/package-data-downloads\/ttf-mscorefonts-installer
\nsudo \/usr\/lib\/update-notifier\/package-data-downloader<\/p>\n

Obviously put your password when requested even if you do not see the typing of the same !!
