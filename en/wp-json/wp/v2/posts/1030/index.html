\nafter several attempts I present a new script for Nautilus (file manager present in most linux distributions): allows you to rename all the PDF files, within the same folder, according to the internal tag of the title (for instance right click, property, document, title); very useful in various cases !!!
\nfirst of all download this file:
\n<\/p>\n

move the file to
\n\/home\/YourName<\/strong>\/.local\/share\/nautilus\/scripts\/
\nthen by right clicking on the file
\ngo to property > permissions
\nand put the check mark on “Allow file execution as a program”
\nNow just select the file or pdf files that interest you and right click
\nscript > renam-Port-title-tag-in-name.sh
\nNB: ATTENTION<\/strong> the script renames all the PDFs contained in that folder
\nNB: ATTENTION<\/strong> If the PDF file has no title tag the PDF will be deleted
