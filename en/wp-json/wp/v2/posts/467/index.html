\nDownload the following packages from the web (google is your friend):<\/p>\n

calibre_0.9.2+dfsg-1~ppa1_all.deb
\ncalibre-bin_0.9.2+dfsg-1~ppa1_i386.deb
\nfontconfig-config_2.9.0-7_all.deb
\nlibfontconfig1_2.9.0-7_i386.deb

\nlibtiff5_4.0.2-1ubuntu2_i386.deb
\npython-qt4_4.9.3-4_i386.deb<\/p>\n

\nnow digit
\ndpkg -i –force-all *.deb<\/p>\n

Well now we have to keep Calibre We add the ppa<\/p>\n

sudo add-apt-repository ppa:n-muench\/calibre -y<\/p>\n

We will now modify the pope just added:<\/p>\n

sudo gedit \/etc\/apt\/sources.list.d\/n-muench-calibre-precise.list<\/p>\n

Now we’re going to replace “precise” putting “quantal”<\/p>\n

Save and close<\/p>\n

sudo apt-get update
