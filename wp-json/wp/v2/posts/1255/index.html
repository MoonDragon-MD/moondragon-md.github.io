\nSalve, mi sono imbattuto in un mio amico che ha l’iphone con il problema di aprire con Ubuntu le sue foto che per\u00f2 son in formato HEIC.<\/p>\n

Ho trovato una facile soluzione:<\/p>\n

aprire un terminale e scriverci<\/p>\n

sudo apt-get install libheif-examples<\/p>\n

date invio e digitate la vostra password per installare questa dipendenza,<\/p>\n

Dopodich\u00e8 create un file di testo e chiamatelo come preferite, io ho messo “convertire_HEIC_in_JPG.sh”<\/p>\n

dategli i permessi di esecuzione (clic destro propriet\u00e0 > permessi > esecuzione)<\/p>\n

poi apritelo e scriveteci dentro:<\/p>\n

#!\/bin\/bash<\/em>
\nfor f in *.HEIC<\/em>
\ndo<\/em>
\necho “Sto convertendo il file $f”<\/em>
\nheif-convert $f $f.jpg<\/em>
\ndone<\/em><\/p>\n

Ora salvate e spostate il file in<\/p>\n

\/home\/TUONOME\/.local\/share\/nautilus\/scripts<\/p>\n

(al posto di TUONOME\u00a0 mettere il vostro nome utente)<\/p>\n

ora quando andate in una cartella con le foto in HEIC\u00a0\u00a0 basta selezionare tutto,<\/p>\n

fare clic destro > script > convertire_HEIC_in_JPG.sh<\/p>\n

ora partir\u00e0 da sola la conversione, l’attesa dipender\u00e0 dal numero delle foto selezionate!!<\/p>\n

Spero possa esservi d’aiuto<\/p>\n
