nb:Il procedimento pu\u00f2 essere eseguito solo da distro ubuntu<\/p>\n

    \n
  1. Scaricare le iso linux live delle distro che vi piacciono<\/li>\n

    <\/p>\n

  2. Installare UNETBOOTIN (sudo apt-get install unetbootin)<\/li>\n

    <\/p>\n

  3. Inserire pennetta USB nel computer<\/li>\n

    <\/p>\n

  4. Installare una iso nella pennetta con unetbootin (ubuntu va installato per ultimo)<\/li>\n

    <\/p>\n

  5. Aprire la pennetta e creare una cartella con il nome della distro installata<\/li>\n

    <\/p>\n

  6. Trascinare tutti i file della pennetta nella cartella creata<\/li>\n

    <\/p>\n

  7. Ora installare la seconda distro nella pennetta ripetere passaggi da 4 a 7<\/li>\n

    <\/p>\n

  8. Ora andate su sistema > amministrazione > creatore dischi di avvio<\/li>\n

    <\/p>\n

  9. Selezionate la iso di ubuntu e impostate se volete la persistenza<\/li>\n

    <\/p>\n

  10. Aprite la cartella della prima distro creata e aprite il file syslinux.cfg<\/li>\n

    <\/p>\n

  11. Copiare le ultime righe riguardanti il kernel (label ecc)<\/li>\n

    <\/p>\n

  12. Ora aprire il file (in sottocartella syslinux) txt.cfg ed incollateci le righe copiate ,in fondo<\/li>\n

    <\/p>\n

  13. Aprite la cartella della seconda distro creata e aprite il file syslinux.cfg
    \nfate i passaggi 12 e 13<\/li>\n

    <\/p>\n

  14. Se per caso avete puppy dopo ‘label puppy’ > ‘kernel vnlinux’ va trasformato in ‘kernel \/vnlinux’
    \ne anche intrd=\/initr.gz (insomma le strighe del kernel)<\/li>\n

    <\/p>\n

  15. Ora copiare il file del 1 e 2 distro nella \/ della pennetta se vi chiede di unire fatelo ma non sovrascrivete nessun file<\/li>\n

    <\/p>\n

  16. Ora potete fare il boot da usb (impostando prima il bios del pc)<\/li>\n

    <\/p>\n

  17. Si aprir\u00e0 il men\u00f9 boot di ubuntu ma noterete che in fondo alla lista ci saranno le altre distro<\/li>\n

    \n<\/ol>\n
