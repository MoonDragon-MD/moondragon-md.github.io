Salve a tutti oggi vi propongo un’estenzione di nautilus che semplifica la scanzione antivirus su linux.
Vi chiederete ora cosa serve un antivirus su linux :
a) Per scanzionare i file prima di trasferirli su Windows (prevenzione)
b) Per scanzionare un hard-disk infetto (cura)
Lo script da me creato ha un iterfaccia grafica che vi seguir\u00e0 passo per passo prima e durante la scanzione.
La scanzione \u00e8 basata sull’ antivirus per linux ClamAV
Se non abbiamo installato l’antivirus lo script vi proporr\u00e0 l’installazione.

Istruzioni:
1) scaricare lo script da questo link<\/a>
2) Copiare lo script in
vostraHOME\/.gnome2\/nautilus-scripts
(per vedere le cartelle nascoste utilizzare ctrl+h)
3) Dategli i permessi di esecuzione facendo clic destro sul file, propriet\u00e0,permessi,selezionare l’esecuzione come programma.
4) Posizionatevi con il maus su qualunque cartella o file da scanzionare, poi fare clic destro,script,Scanzione-Antivirus

5) Ora vi verr\u00e0 chiesto di aggiornare le firme antivirali (consigliato)

ecco la finestra che comparir\u00e0 durante l’aggiornamento

6) Seguire le istruzioni grafiche e buona scanzione
