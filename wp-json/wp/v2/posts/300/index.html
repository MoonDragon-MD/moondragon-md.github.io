Salve oggi spiegher\u00f2 come avviare firefox con impostazioni personali diversificate e separate<\/p>\n

Questo potrebbe essere utile quando:
\n1) <\/span>Si vorrebbe provare una nuova versione (pre o beta) di firefox
\n2)<\/span> Si vuole tenere una configurazione pulita e veloce mantenendo l’altra personalizzata con plugin e varie<\/p>\n

ISTRUZIONI :<\/span><\/p>\n

nb<\/span>:Volendo scaricare anche una nuova versione andare qui<\/p>\n

Clicca qui per Scaricare Firefox<\/a><\/p>\n

Allora prima di tutto chiudete firefox (quindi vi consiglio di stampare la pagina)<\/p>\n

1)<\/span> clic contemporaneo “alt+F2”<\/p>\n

scrivere:\u00a0\u00a0\u00a0\u00a0\u00a0\u00a0\u00a0\u00a0\u00a0\u00a0\u00a0\u00a0\u00a0\u00a0\u00a0\u00a0\u00a0\u00a0\u00a0\u00a0 firefox –profilemanager\u00a0\u00a0\u00a0\u00a0\u00a0\u00a0\u00a0\u00a0\u00a0\u00a0 ; dare invio<\/p>\n

Si aprir\u00e0 una scermata dove andremo ad aggiungere un nuovo profilo ; es:\u00a0\u00a0\u00a0 firefox-beta<\/p>\n

<\/p>\n

Volendo possiamo rinominare il profilo default<\/p>\n

es: Fox\u00a0\u00a0\u00a0\u00a0\u00a0\u00a0\u00a0\u00a0 ; salvare e chiudere<\/p>\n

2)<\/span> Ora andremo a modificare il lanciatore di firefox nel men\u00f9<\/p>\n

clic destro su ; applicazioni > modifica men\u00f9<\/p>\n

ora andiamo nella sezione internet e sulla voce “firefox” facciamo un clic<\/p>\n

una volta selezionato clicciamo a destra\u00a0 “Modifica”<\/p>\n

nella sezione comando cancelliamo tutto (firefox %u)<\/p>\n

mettendoci:\u00a0\u00a0\u00a0 firefox -P Fox\u00a0\u00a0\u00a0\u00a0\u00a0\u00a0\u00a0\u00a0\u00a0\u00a0\u00a0\u00a0\u00a0 ; clic su chiudi<\/p>\n

nb<\/span>: al posto di Fox andremo a mettere il nome del profilo di default o che avevamo rinominato<\/p>\n

3)<\/span> ora andremo a fare clic su “nuova voce”<\/p>\n

e andiamo a mettere :<\/p>\n

A<\/span>: per chi non ha un altro firefox nel sistema<\/p>\n

nome = Firefox Veloce<\/p>\n

comando =\u00a0 firefox -P Fox-veloce<\/p>\n

nb<\/span> : al posto di “Fox-veloce” andremo a mettere il secondo profilo che abbiamo inserito prima<\/p>\n

B:<\/span> per chi ha due firefox (uno installato di default e l’altro scaricato a mano)<\/p>\n

nome = Firefox beta<\/p>\n

comando =\u00a0 \/var\/firefox\/firefox -P Firefox-beta<\/p>\n

nb<\/span> : al posto di “\/var\/firefox\/” andremo a mettere il percorso cartella dove abbiamo estratto il file .tar di firefox che abbiamo scaricato all’inizio ; al posto di “Firefox-beta” andremo a mettere il secondo profilo che abbiamo inserito prima<\/p>\n

chiudere le finestre aperte<\/p>\n

BENE ORA SIETE PRONTI<\/span><\/p>\n

Ora in applicazioni > internet troveremo due tipi di firefox<\/p>\n

Mi raccomando non si possono avviare tutti e due contemporaneamente<\/p>\n

quindi chiuso uno potremo utilizzare l’altro.<\/p>\n

Buona navigazione “personalizzata”<\/p>\n
