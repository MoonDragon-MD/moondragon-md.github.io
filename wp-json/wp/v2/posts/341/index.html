Dopo aver fatto una ricerca in rete ho constatato che la compressione dei file pdf \u00e8 abbastanza oscura!!!<\/p>\n

Per semplificarvi la vita ho creato uno script Nautilus (testato sulle versioni 10.04 e 11.04 di ubuntu 64bit,ma funzionante con tutte)<\/p>\n

sudo apt-get install ghostscript<\/strong><\/p>\n

Ora scaricate lo script da me creato
\n<\/p>\n

Una volta scaricato lo script dovr\u00e0 essere messo in<\/p>\n

\/home\/vostronome<\/span>\/.gnome2\/nautilus-scripts<\/strong><\/p>\n

poi facendo clic destro sul file<\/p>\n

andate su propriet\u00e0 > permessi<\/p>\n

e mettete la spunta su “Consentire l’esecuzione del file come programma”<\/p>\n

script > Comprimi-PDF<\/p>\n
